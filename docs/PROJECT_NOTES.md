# Project Notes

## 设计目标

月白历优先做一件事：在 macOS 菜单栏里提供一个颜色自适应、轻量、原生的农历日历。

## 为什么读取系统日历

中国大陆法定节假日和调休日每年都有变化。如果硬编码进 App，需要持续维护，也容易出错。macOS 自带日历已经可以订阅官方或系统提供的节假日日历，所以月白历直接读取用户已有日历订阅。

## 当前限制

- 只识别标题包含“节假日 / 假日 / holiday”的日历
- 只显示全天事件
- 没有独立设置窗口，设置放在右键菜单
- 暂未提供自动更新

## 未来方向

- 图标和命名继续打磨
- 支持更多语言
- 支持选择具体节假日日历
- 支持用户自定义弹窗宽度和菜单栏格式
- 增加截图和 Homebrew Cask 发布流程
- 考虑迁移到 Swift Package 或 Xcode project，以便后续多人维护
